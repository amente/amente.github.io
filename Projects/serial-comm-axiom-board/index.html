--- layout: page title: "Serial Communication with Axiom Board" description: "Done as a bonus course work for SYSC2003 at Carleton University" group: project worked: team --- {% include JB/setup %} As part of a course work for SYSC 2003 (Introductory Real Time Systems), with my partner [@excsniper](https://github.com/excsniper) we worked on a debugger applcation for a simple robot system based on Axiom Project Developement Board (Motorolla HCS12 MCU based). The hardware side of the code was written in C and cross compiled using ICC12. We also developed a simple JAVA desktop application to control the robot via a serial port using our own protocol
